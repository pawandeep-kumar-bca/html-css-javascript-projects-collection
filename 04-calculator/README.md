# 🧮 Calculator App

A basic calculator built using HTML, CSS, and JavaScript.
Performs arithmetic operations with a clean UI.

---

## 🚀 Features

* Addition, Subtraction, Multiplication, Division
* AC (clear all)
* DE (delete last digit)
* Decimal support
* Instant calculation

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript

---

## 📂 Project Structure

calculator-app/
│
├── index.html
├── style.css
└── README.md

---

## 💡 How It Works

1. Click buttons
2. Expression builds
3. Press "=" to get result

---

## ▶️ How to Run

1. Clone repo
   git clone https://github.com/your-username/calculator-app.git

2. Open index.html
