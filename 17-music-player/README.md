# Music Player Web App

A simple and responsive Music Player built using HTML, CSS, and JavaScript.  
It allows users to play, pause, and control audio with a clean user interface.

---

## Features

- Play and Pause functionality
- Song progress tracking
- Seek control using range slider
- Real-time progress update
- Clean and responsive UI

---

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

## How to Run

1. Download or clone the project
2. Open the project folder
3. Add your own audio file in place of the existing one
4. Open index.html in your browser

---

## Usage

- Click on Play button to start music
- Click again to Pause
- Use the progress bar to seek the audio
