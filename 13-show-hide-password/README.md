# 👁️ Hide & Show Password Toggle

A simple and interactive **Password Toggle UI** built using **HTML, CSS, and JavaScript**.  
Users can show or hide their password by clicking on the eye icon.

---

## 🚀 Features

- 🔐 Password input field
- 👁️ Toggle visibility (Show / Hide password)
- ⚡ Instant UI response using JavaScript
- 🎨 Clean and minimal design
- 📱 Responsive layout

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (DOM Manipulation)

## ⚙️ How It Works

- Initially password is hidden (`type="password"`)
- When user clicks on eye icon 👁️:
  - Input type changes to `text`
  - Icon changes to "show"
- Clicking again:
  - Input type switches back to `password`
  - Icon changes to "hide"
