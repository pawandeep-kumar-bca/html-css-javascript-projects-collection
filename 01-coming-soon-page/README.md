# 🚀 Coming Soon Website

A modern and responsive **Coming Soon / Maintenance Page** with a live countdown timer and animated UI. This project is built using **HTML, CSS, and JavaScript**.

---

## ✨ Features

* ⏳ Live Countdown Timer
* 🎨 Clean & Modern UI Design
* 🚀 Animated Rocket Effect
* 📱 Fully Responsive Layout
* 🔥 Easy to Customize Launch Date

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox + Animations)
* JavaScript (DOM + Timer)

## ⚙️ How It Works

* A fixed **launch date** is set in JavaScript.
* The script calculates the difference between the current time and the launch date.
* It updates the countdown every second.

```javascript
let countDownDate = new Date("Oct 16, 2025 00:00:00").getTime();
```
