# Popup Modal Project

A simple popup modal built using HTML, CSS, and JavaScript.  
It displays a success message when the user clicks the submit button.

---

## Features

- Open popup on button click
- Close popup on OK button click
- Smooth animation using CSS transitions
- Centered and clean UI design

---

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

---

## Project Structure

18-popup-modal/
│
├── index.html
├── style.css
├── app.js
└── check.png

---

## How to Run

1. Download or clone the project
2. Make sure check.png is present in the folder
3. Open index.html in your browser

---

## Usage

- Click Submit button to open popup
- Click OK button to close popup

---

## Code Logic

- openPopup() → Adds class to show popup
- closePopup() → Removes class to hide popup
