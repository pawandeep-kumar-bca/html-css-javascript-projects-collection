# 🔐 Random Password Generator

A simple and responsive **Random Password Generator Web App** built using **HTML, CSS, and JavaScript**.
It generates strong, secure passwords with a mix of uppercase, lowercase, numbers, and symbols.

---

## 🚀 Features

* 🔑 Generate strong random passwords
* 🔤 Includes uppercase, lowercase, numbers, and symbols
* 📋 One-click copy to clipboard
* ⚡ Fast and lightweight
* 🎨 Clean and responsive UI

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## ⚙️ How It Works

* Click on **"Generate Password"**

* A random password of length **12** is created

* Password includes:

  * Uppercase letters (A-Z)
  * Lowercase letters (a-z)
  * Numbers (0-9)
  * Symbols (!@#$...)

* Click the **copy icon** to copy password to clipboard

---

## ▶️ How to Run

1. Download or clone the repository
2. Open the project folder
3. Run `index.html` in your browser

---

## 🧠 Logic Overview

* Random characters are selected using `Math.random()`
* Ensures at least:

  * 1 uppercase
  * 1 lowercase
  * 1 number
  * 1 symbol
* Remaining characters are filled randomly
