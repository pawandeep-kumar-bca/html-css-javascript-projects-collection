# 🔵 Circular Progress Bar

A beautiful animated circular progress bar built using **HTML, CSS, and JavaScript**.
It visually represents progress percentage with smooth animation.

---

## 🚀 Features

* 📊 Animated circular progress bar
* 🎨 Gradient stroke design
* ⚡ Smooth percentage animation
* 💡 Clean neumorphic UI
* 📱 Fully responsive

---

## 🛠️ Tech Stack

* HTML5 (SVG + Structure)
* CSS3 (Animation + Shadows + Gradient)
* JavaScript (Counter logic)

## 💡 How It Works

1. SVG circle is used to create progress ring
2. `stroke-dasharray` defines total length
3. `stroke-dashoffset` controls progress
4. JavaScript increments percentage (0 → 100)
5. CSS animation fills the circle smoothly
