# 📅 Mini Calendar

A simple and elegant Mini Calendar built using **HTML, CSS, and JavaScript**.
It dynamically displays the current date, day, month, and year.

---

## 🚀 Features

* 📆 Displays current date
* 📅 Shows day (Sunday–Saturday)
* 🗓️ Shows month and year
* ⚡ Auto-updates using JavaScript
* 🎨 Clean and modern UI
* 📱 Responsive design

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox + Gradient UI)
* JavaScript (Date Object)

---

## 📂 Project Structure

mini-calendar/
│
├── index.html

---

## 💡 How It Works

1. JavaScript gets current date using `Date()`
2. Extracts:

   * Day
   * Date
   * Month
   * Year
3. Updates UI dynamically
