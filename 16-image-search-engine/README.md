# 🖼️ Image Search Engine

A simple and responsive Image Search Engine built using **HTML, CSS, and JavaScript**.
This project uses the **Unsplash API** to fetch and display images based on user search queries.

---

## 🚀 Features

* 🔍 Search any image using keywords
* 🖼️ Displays results in a grid layout
* ➕ "Show More" button to load more images
* 📱 Fully responsive design
* ⚡ Fast and dynamic API-based search

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Unsplash API

## 📸 How It Works

1. User enters a keyword in the search box
2. API request is sent to Unsplash
3. Images are fetched and displayed
4. Clicking "Show More" loads additional images
