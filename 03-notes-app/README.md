# 📝 Notes App

A simple Notes App where users can create, edit, and delete notes.
All data is stored in browser localStorage.

---

## 🚀 Features

* Create notes
* Edit notes
* Delete notes
* Auto-save with localStorage
* Data persists after refresh

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript

## 💡 How It Works

1. Click "Create Notes"
2. Type inside note
3. Notes auto-save
4. Delete anytime
