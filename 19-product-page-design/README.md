# 🛍️ Product Page Design

A modern and responsive **Product Page UI** built using **HTML, CSS, and JavaScript**.
This project showcases an e-commerce product page with image gallery, size & color selection, and quantity control.

---

## 🚀 Features

* 🖼️ Product Image Gallery (with clickable thumbnails)
* 🎨 Color Selection UI
* 📏 Size Selection Options
* 🔢 Quantity Input
* 💸 Pricing with Discount Display
* ⚡ Interactive UI using JavaScript
* 📱 Responsive Layout

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## ⚙️ How It Works

* Clicking on the small dots (buttons) changes the product image.
* Selected button gets highlighted using the `active` class.
* Size and color selections use radio buttons with custom styling.

Example logic:

```javascript
btn[0].onclick = function(){
    productImg.src = "./image1.png";
}
```
