# 💰 Cryptocurrency Website

A modern Cryptocurrency landing page built using **HTML, CSS, and JavaScript**.
It displays live crypto prices using an API and provides a clean UI for users.

---

## 🚀 Features

* 💸 Live cryptocurrency prices (Bitcoin, Ethereum, Dogecoin)
* 🌐 API integration (CoinGecko)
* 🎨 Modern landing page design
* 📱 Responsive layout
* ⚡ Real-time data fetching

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox + Background Image + UI Design)
* JavaScript (Fetch API + Async/Await)

## 💡 How It Works

1. JavaScript fetches data from CoinGecko API
2. API returns latest crypto prices
3. Prices are updated dynamically in UI
4. Users see real-time values
